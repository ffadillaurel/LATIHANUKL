Project ini dibuat untuk latihan persiapan UKL semester ganjil. restful API ini mensimulasikan sistem presensi online
yang memiliki fitur:
- Login pengguna menggunakan JWT
- Pengelolaan data pengguna (menambah, menampilkan)
- Pengelolaan data presensi / attendance
- Menampilkan riwayat presensi berdasarkan user
- Melihat ringkasan presensi (summary)
- Analisis kehadiran berdasarkan parameter tertentu
